<h1 align="center">Side Project Insightify</h1>
<h3 align="center">Insightify is a SaaS platform designed to streamline the feedback collection process. It allows creators to build custom forms and share them with a simple link. The key feature is a frictionless experience for respondents, who can submit feedback instantly without any mandatory sign-up, leading to higher participation rates.</h3>

- Example Public board [SideSync Board](https://www.insightify.cam/b/67c1b139917f32a3dbd306f1)
